# 🚀 Astra Yield Protocol

A simple DeFi staking protocol built on Ethereum compatible smart contracts.

---

## 📌 Overview
Astra Yield Protocol allows users to stake ERC20 tokens and earn fixed yield rewards through a decentralized smart contract system.

---

## ⚙️ Features
- ERC20 Token (ASTRA)
- Staking Vault
- Fixed APR reward system (10%)
- Simple and secure DeFi architecture

---

## 🔄 Flow
User → stake ASTRA → tokens locked → rewards calculated → withdraw anytime

---

## 🧱 Tech Stack
- Solidity ^0.8.20
- OpenZeppelin Contracts
- Remix IDE
- Ethereum compatible network

---

## 🌐 dApp Interface
Frontend allows users to:
- Connect wallet
- Stake tokens
- Withdraw tokens
- Check rewards

---

## 🚀 Purpose
This project demonstrates core DeFi primitives including staking, token utility, and yield generation logic.
